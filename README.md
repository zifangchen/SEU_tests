# SEU_tests
东南大学课程与考试资料整理

本项目完全开源，完全无盈利性质。禁止以各种盈利为目的传播。
仅个人整理，不代表任何官方，侵权请联系213220049@seu.edn删除。

本项目旨在消除SEUer之间的“信息差”，或者说减少SEUer为了消除信息差而花费的许多无谓的时间。
希望学弟学妹们把时间花在真正有意义的事情上，而不是应付考试。
