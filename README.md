## 📋 项目说明
<div style="background-color: #fff8e1; border-left: 4px solid #FFA000; padding: 15px; border-radius: 5px; margin: 20px 0;">
  <strong>⚠️ 重要声明</strong><br>
  本项目完全开源，完全无盈利性质。禁止以各种盈利为目的传播。<br>
  仅个人整理，不代表任何官方，侵权请联系 <a href="mailto:213220049@seu.edu.cn">213220049@seu.edu.cn</a> 删除。
</div>

<div style="background-color: #e8f5e9; border-left: 4px solid #4CAF50; padding: 15px; border-radius: 5px; margin: 20px 0;">
  <strong>🎯 项目宗旨</strong><br>
  本项目旨在消除SEUer之间的"信息差"，或者说减少SEUer为了消除信息差而花费的许多无谓的时间。<br>
  希望学弟学妹们把时间花在真正有意义的事情上，而不是应付考试。
</div>
